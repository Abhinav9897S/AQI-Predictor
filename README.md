# AQI Predictor

## Overview
Machine learning-based system for predicting AQI values.

## Features
- Data preprocessing
- AQI prediction
- Data visualization
- Model evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Dataset
Noida AQI dataset covering 3 years.
